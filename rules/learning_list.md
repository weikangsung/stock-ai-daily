# 規則學習清單

*此檔案記錄所有待驗證的新發現規則。規則驗證成功後會升級到 CLAUDE.md 的「主規則庫」。*

---

## 使用說明

### 規則狀態流轉
1. **待驗證 (0/10)**：新發現的規則，開始累計驗證次數
2. **驗證中 (X/10)**：持續驗證中，每次收盤後更新
3. **準備升級 (7+/10)**：驗證成功率 ≥ 70%，等待用戶確認升級
4. **已升級**：規則升級至 CLAUDE.md 主規則庫，移出此檔案

### 何時添加規則
- 每次收盤檢討時，若發現新的交易規律，就添加新規則
- 規則應該具體、可量化、可重複驗證

### 規則編寫範例

```markdown
## 規則 #001 - 美股漲幅帶動台股

- **規則內容**：美股昨晚漲幅 > 1% 時，正2當天有較高上漲概率
- **觸發條件**：S&P500 昨晚漲幅 > 1%
- **交易方向**：做多 / 看漲
- **發現日期**：2026-05-29
- **發現背景**：首次檢討發現美股與台股同向性較強

**驗證狀態**：待驗證 (0/10)

**驗證記錄**：

| 日期 | 條件是否觸發 | 預測方向 | 實際結果 | 符合 | 備註 |
|------|-------------|---------|---------|------|------|
|      |             |         |         |      |      |
```
